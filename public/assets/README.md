# Assets

Static files in `public/` are served from the site root.

- **images/** – Photos, hero images, project thumbnails. Use in Hero image tiles or anywhere: `/assets/images/hero-1.jpg`
- **icons/** – Logos, favicons, icons. Use in Nav or elsewhere: `/assets/icons/logo.svg`

Add your files here and reference them by path (e.g. `/assets/images/photo.jpg`).
